Class Hierarchy

  • appdir.AppDir: An AppDir object is an object representation of https://apps.mozillalabs.com/appdir The purpose of the object is to be able to dissect the page and determine what is installed
  • appobject.AppObject: AppObject tries to encapsulate the visual representation for an app on apps.mozillalabs.com/appdir page It breaks down an app into a name, install_icon, nameregion, icon..etc
  • firefox.Firefox: This class encapsulates actions that occur on the browser
  • myapps.MyApps: MyApps class creates a page object model
  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • firefox.TimeoutStartupError: Thrown to indicate that firefox failed to startup within a time period.
        • constructbox.UnsuppportedOSError
        • constructbox.WinError: Error to indicate a windows-specific issue, such as finding windows-specific icons.
    • constructbox.Box: Base class representation for all machines
      • constructbox.LinBox: linuxbox is currently untested, but will need to be filled in when appropriate
      • constructbox.MacBox: A MacBox object will contain functions that are mac specific
      • constructbox.WinBox: A windows box will contain functions that are windows specific
    • constructbox.ConstructBox: A call to the ConstructBox class returns an object that is tailored to return operating system dependent data
