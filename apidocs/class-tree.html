Class Hierarchy

  • appdir_page.AppDirPage: An AppDir object is an object representation of https://apps.mozillalabs.com/appdir.
  • app_object.AppObject: AppObject encapsulates the visual representation for an app on the https://apps.mozillalabs.com/appdir page.
  • firefox.Firefox: This class encapsulates actions that occur on the Firefox browser.
  • myapps_page.MyAppsPage: Represents the My Applications Web Page at the below URL specified in the constant.
  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • myapps_page.ApplicationNotFoundError: Thrown to indicate that the application image specified was not found.
        • firefox.TimeoutStartupError: Thrown to indicate that firefox failed to startup within a time period.
        • os_box.UnsuppportedOSError: Thrown to indicate that the OS running is not supported.
        • os_box.WinError: Error to indicate a windows-specific issue, such as finding windows-specific icons.
    • os_box.ConstructOSBox: Factory class used to generate the appropriate OSBox object based on the underlying operating system being used.
    • os_box.OSBox: Base class representation for all machines
      • os_box.LinOSBox: linuxOSBox is currently untested, but will need to be filled in when appropriate
      • os_box.MacOSBox: A MacOSBox object will contain functions that are mac specific
      • os_box.WinOSBox: A windows OSBox will contain functions that are windows specific
