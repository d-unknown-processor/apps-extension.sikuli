Class AppObject

source code

Method Details [hide private]

topleft(self, name, region)

source code 

Attempts to set x, and y coordinates of the application, based upon the region you are passing in. If you have the "Install Button" then you can work backwards and find the top left for the application object. Once you have the top left, all the other positioning is known.

iconimage(self)

source code 

Stores the region image in a file, and returns the location. This technique is good when you want to compare images across multiple pages
